# RIKA Lens Calculator

Калькулятор портфеля объективов для тепловизионных прицелов.

Подбирает набор стандартных фокусных расстояний для всей продуктовой линейки приборов с разными сенсорами и дисплеями. Минимизирует ошибку округления px/mrad — «плавание» штрихов прицельной сетки.

**Live:** https://shah1git.github.io/rika-lens-calc/

## Стек

- Vite + React + TypeScript
- GitHub Pages (деплой через GitHub Actions)
- Без CSS-фреймворков — inline styles + минимальный `global.css`

## Структура

```
src/
  App.tsx       — главный компонент: состояние, computed, JSX
  i18n.ts       — переводы RU/EN/ZH (объект T)
  optics.ts     — интерфейсы + чистые функции (calcAxis, getScore, findMultiples)
  ui.tsx        — палитра, пресеты, UI-компоненты (Cd, TH, Nm, Sel...)
  main.tsx      — точка входа
  global.css    — reset + CSS-тултипы для таблицы
index.html      — подключение JetBrains Mono, lang=ru
vite.config.ts  — base: /rika-lens-calc/
```

## Разработка

```bash
npm install
npm run dev     # dev-сервер
npm run build   # сборка в dist/
```

## Деплой

Автоматический через `.github/workflows/deploy.yml` при пуше в `main`.

## Функциональность

- **Пресеты сенсоров:** 256x192, 384x288, 640x480, 640x512, 1024x768, 1280x1024
- **Пресеты дисплеев:** 640x480, 1024x768, 1280x1024, 1920x1080, 2560x2560
- **Шаг пикселя:** 12, 15, 17, 25 µm
- **Диапазон фокусных:** 5–200 мм, общий для всех конфигураций
- **i18n:** RU / EN / ZH, переключатель флагами, сохранение в localStorage
- **Конфигурации приборов:** до 6 (мин. 2), каждая — сенсор + шаг + дисплей + имя
- **Приоритет осей:** «Только V» / «Приоритет V»
- **Таблица результатов:** сортировка по worst case / покрытию / отдельной конфигурации
- **Портфель объективов:** клик по строке добавляет/убирает фокусное; матрица ошибок и покрытие
- **URL sharing:** все параметры сохраняются в hash, кнопка «Скопировать ссылку»

## Версионирование

Версия отображается в заголовке рядом с названием (`v3.0`, `v3.1`, ...).
При каждом коммите версия инкрементируется:
- **patch** (3.0 → 3.1) — баг-фиксы, мелкие правки
- **minor** (3.x → 4.0) — новые блоки, фичи, существенные изменения
